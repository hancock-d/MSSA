3.1d Array

Write a function which takes an array as input
and finds the first occurrence of 2 consecutive 1s
	and changes their value to 0.
if you have two occurrences of 1, change their values to 0 and come out of the function

For e.g: Input : [0,2,1,1,9,1,1]

Output: [0,2,0,0,9,1,1]

Set the array size to 6
do not use string.Join
