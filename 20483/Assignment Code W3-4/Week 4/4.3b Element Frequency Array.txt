4.3b Element Frequency Array

Write a program in C# Sharp to count the frequency of each element of an array.
Hardcode the elements first.
Then get the elements from the user.

Test Data :
Input the number of elements to be stored in the array :3
Input 3 elements in the array :
element - 0 : 25
element - 1 : 12
element - 2 : 43
Expected Output :
Frequency of all elements of array :
25 occurs 1 times
12 occurs 1 times
43 occurs 1 times


Unique element = single occurrence
Duplicate element = More than one occurrence


Can use any other data structure like a dictionary or hash set 5 Feb @ 2h34m
